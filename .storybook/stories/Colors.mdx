# Colors

This UI extends default [tailwindcss color palette](https://tailwindcss.com/docs/customizing-colors) with additional colors.

You can combine colors to set text, background, border, ring, and other properties.

Examples:

* <div className="text-primary">text-primary</div>
* <div className="text-primary-foreground bg-primary">text-primary-foreground bg-primary</div>
* <div className="border border-primary">border border-primary</div>
