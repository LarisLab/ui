# Installation

## 1. Install package

```
npm i @larislab/ui tailwindcss-animate lucide-react
```

## 2. Add plugins to `tailwind.config.ts` / `tailwind.config.mjs`

```ts
export default {
    plugins: [require('tailwindcss-animate'), require('@larislab/ui/plugin')],
}
```

That's it! You're ready to use the components.
